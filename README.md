# Color-Pallette-For-Indian-Marketing-Events
a professional-grade design system utility built for Indian Marketing Teams. It moves beyond simple color picking by using mathematical harmonies and researched cultural palettes to create cohesive, accessible, and  production-ready themes.

 **Harmony Lab:** Generate themes based on **Complementary, Triadic, Analogous, and Split-Complementary** logic.
   10 - **Gradient Lab:** Create high-fidelity, multi-stop linear gradients with a single click.
   11 - **Cultural Intelligence:** 50+ researched presets for events like **Diwali, Holi, Christmas, Eid, Onam, and more**, ensuring
      visual significance and accuracy.
   12 - **1-Click Figma Export:** Instant SVG generation that copies directly to your clipboard. Paste into Figma for a 1:1 design
      system board.
   13 - **Accessibility First:** Integrated WCAG AA (4.5:1) contrast checking logic ensures every generated theme is readable.
   14 - **Interactive Library:** Save your favorite harmonies and gradients to a local archive (LocalStorage) for future sessions.
   15 - **Cinematic UI:** High-performance interactive particle backgrounds and a sleek, aero-glass aesthetic.

   ## 🛠️ Tech Stack
   21 - **Frontend:** Vanilla JavaScript (ES6+), HTML5, CSS3.
   22 - **Styling:** Tailwind CSS + Custom Vanilla CSS for Aero effects.
   23 - **Graphics:** Canvas API for interactive 2D matrix/flow animations.
   24 - **Storage:** Browser LocalStorage for persistent libraries.

   **Generate:** Choose a festival or "Random" and hit **Mix New Theme**.
   31 2. **Refine:** Toggle between the Harmony Lab and Gradient Lab.
   32 3. **Save:** Use the floating bookmark icon to store a palette in your local library.
   33 4. **Export:** Click **Export to Figma**, then go to your Figma canvas and press `Cmd+V` (or `Ctrl+V`). You'll get a fully
